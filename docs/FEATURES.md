## ‚ö° Hardware-Aware Performance

### Automatic Detection
- CPU cores and capabilities
- RAM amount and speed
- Storage type (SSD/HDD)
- GPU vendor (AMD/Intel/Nvidia)
- NPU presence for AI acceleration

### Performance Profiles

#### High-End Systems (16GB+ RAM, 8+ cores, SSD)
- **Performance CPU governor**: Maximum clock speeds
- **Minimal swap**: RAM-first approach
- **Aggressive I/O**: Optimized for NVMe/SSD
- **Maximum GPU performance**: No power limits

#### Balanced Systems (8-16GB RAM, 4-8 cores)
- **Schedutil governor**: Intelligent frequency scaling
- **Moderate swap**: Balanced memory management
- **BFQ I/O scheduler**: Fair queue management
- **Balanced GPU**: Power and performance

#### Efficiency Systems (<8GB RAM, <4 cores)
- **Powersave governor**: Energy efficient
- **Zram compression**: Virtual swap in RAM
- **Conservative I/O**: Battery-friendly
- **Power-optimized GPU**: Extended battery life

### GPU Support
- **NVIDIA**: Full driver stack, CUDA, acceleration
- **AMD**: Mesa, ROCm, Vulkan
- **Intel**: VA-API, Quick Sync, integrated graphics
- **Multi-GPU**: Automatically configured

### NPU Support
- Intel AI accelerators (Meteor Lake+)
- AMD XDNA NPU
- OpenVINO integration
- AI workload acceleration
